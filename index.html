Integrasi Google Drive & Google Sheets

Hubungkan dashboard studi hukum Anda langsung ke Google Sheets dan Folder Google Drive Anda.

Gunakan kode Google Apps Script yang kami sediakan di panduan bawah untuk membuat Web App pribadi Anda.

Panduan Konfigurasi Database (Langkah demi Langkah)

Aplikasi ini berjalan mandiri di browser Anda menggunakan penyimpanan lokal (LocalStorage). Untuk menyimpan data secara permanen di Google Sheets dan mengupload berkas ke Google Drive secara gratis, ikuti langkah berikut:

  1. Buat Spreadsheet baru di Google Sheets. Beri nama misalnya "JurisDoc Database".
  2. Di dalam Google Sheets, klik menu Ekstensi (Extensions) > Apps Script.
  3. Hapus semua kode bawaan, lalu salin (copy) kode script backend yang telah kami sediakan di bagian luar web ini (dalam instruksi chat).
  4. Klik tombol Simpan / Save (ikon disket).
  5. Klik tombol Terapkan > Penerapan baru (Deploy > New Deployment).
  6. Pilih jenis Aplikasi Web (Web App).
  7. Setel "Yang memiliki akses" (Who has access) menjadi "Siapa saja" (Anyone). Klik Terapkan (Deploy).
  8. Izinkan akses keamanan akun Google Anda (klik Advanced > Go to Untitled Project (unsafe) jika muncul peringatan).
  9. Salin URL Aplikasi Web (Web App URL) yang diberikan dan tempelkan (paste) di kolom input di atas!
